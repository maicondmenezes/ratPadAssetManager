## About <a name = "about"></a>

ratPadAssetManager is a Python-based application designed to streamline the management and display of computer hardware assets. Utilizing Python's versatility and its extensive standard library, the project provides an efficient platform for both application logic development and system hardware interaction without external dependencies. It emphasizes simplicity, efficiency, and accessibility, making it an ideal tool for developers interested in system-level programming, open-source development, and practical hardware asset management solutions.

## Getting Started <a name = "getting_started"></a>

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

You need Python installed on your machine. Python 3.x is recommended. No additional external libraries are required as the project uses standard Python libraries.


### Installing

To get a development environment running, clone the repository and navigate to the project directory.

1. Clone the repository

```bash
git clone https://github.com/yourrepository/ratPadAssetManager.git
```


2. Navigate to the project directory
```bash
cd ratPadAssetManager
```


3. Run the application
```bash
python3 main.py
```


Follow the on-screen instructions to navigate through the application.

## Usage <a name = "usage"></a>

Use the system by executing the `main.py` script. The application provides an interface to view and manage computer hardware information, including CPU details and overall system configuration. Further usage instructions and options will be provided within the application's interface.
